Chapter#21 Changing case

  • A human knows that in this context "intermediate" means "Intermediate." But JavaScript doesn't. Because JavaScript is Case Sensitive. We need some way to get JavaScript to recognize the uncapitalized version as a match.
  • There is a method .to UpperCase(); in JavaScript to convert lowercase words to capital words.
  • There is a method .toLowerCase(); in JavaScript to convert Capital words to Lowercase words.
