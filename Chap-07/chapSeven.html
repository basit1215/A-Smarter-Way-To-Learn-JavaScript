Chapter#07 Math expressions: Eliminating ambiguity

  • var num = 1 + 2 * 7;
    Result: num = 15;

  • var num2 = (1 + 2) * 7;
    Result: num2 = 21;

  • var num3 = 1 + (2 * 7);
    Result: num3 = 15;

  • var num4 = (2 + 8) * ( 9 + 5);
    Result: num4 = 140;

  • var num5 = 2 + 8 * 9 + 5;
    Result: num5 = 79 ;

  • var num6 = ((2 + 8) * 9) + 5;
    Result: num6 = 95;
