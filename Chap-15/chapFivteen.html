Chapter#15 Arrays

  • The first item always has an index of 0, not 1.
  • The first item always has an length is 1, not 0.
  • The same naming rules you learned for ordinary variables apply. Only letters, numbers, $ and _ are legal. The first character can't be a number. No spaces.
  • ike an ordinary variable, you declare an array only once. If you assign new values to an array that has already been declared, you drop the var.
  • Arrays can store all data types.
